START HERE - Guide après entraînement

What is already inside this pack:
- model/siamese_change_detector_phase2bis.onnx
- model/siamese_change_detector_phase2bis.onnx.data
- model/deployment_config_phase2bis.json
- backend/inference_onnx.py
- backend/inference_onnx_mqtt.py
- broker/mosquitto.conf
- web/index.html

The model expects:
- image_A = T1, shape [1,3,256,256]
- image_B = T2, shape [1,3,256,256]
- normalization mean=[0.485,0.456,0.406], std=[0.229,0.224,0.225]

Steps:
1) Install Python dependencies:
   pip install -r requirements.txt

2) Put test images:
   test_images/test_01_dataset/T1.png
   test_images/test_01_dataset/T2.png

3) Test inference without MQTT:
   cd backend
   python inference_onnx.py --t1 ../test_images/test_01_dataset/T1.png --t2 ../test_images/test_01_dataset/T2.png --id_test test_01_dataset

4) Start MQTT broker:
   cd ../broker
   mosquitto -c mosquitto.conf -v

5) Open interface:
   web/index.html

6) Run inference + MQTT:
   cd ../backend
   python inference_onnx_mqtt.py --t1 ../test_images/test_01_dataset/T1.png --t2 ../test_images/test_01_dataset/T2.png --id_test test_01_dataset

If interface does not connect, check that Mosquitto is running and that port 9001 websockets is active.
